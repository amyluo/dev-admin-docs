##Pre-condition for install mkdocs

####1. Check python version
```
$ python --version
Python 2.7.15rc1
```
####2. Install pip
```
$ python get-pip.py

$ pip --version
pip 9.0.1 from /usr/lib/python2.7/dist-packages (python 2.7)
```

####3. Installing MkDocs
```
$ pip install mkdocs
$ mkdocs --version
mkdocs, version 1.0.4 from /usr/local/lib/python2.7/dist-packages/mkdocs (Python 2.7)
```
